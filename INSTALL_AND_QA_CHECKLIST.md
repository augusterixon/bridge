# Bridge Install + QA Checklist (v1)

This checklist is used to provision a Pixel into a Bridge device.

---

## Hardware check (before install)
- [ ] Screen OK (no cracks/dead pixels)
- [ ] Buttons work
- [ ] Speaker + mic work
- [ ] USB-C port works
- [ ] Battery health acceptable

---

## Provisioning
- [ ] Factory reset completed
- [ ] Bridge Launcher installed
- [ ] Bridge set as default Home app
- [ ] Device Owner / lockdown applied
- [ ] Play Store browsing disabled or removed
- [ ] Only allowlisted apps present

---

## Core feature QA
- [ ] Home menu matches spec exactly
- [ ] Calls work
- [ ] SMS works
- [ ] Maps opens in route-only mode
- [ ] QR scanner works (camera only here)
- [ ] Hotspot toggle works
- [ ] Library opens local USB files

---

## Exit + restore
- [ ] Exit Bridge flow reachable
- [ ] Factory reset restores normal phone state

---

## Ready to ship
- [ ] Case installed snugly
- [ ] Logo present
- [ ] Device boots directly into Bridge
